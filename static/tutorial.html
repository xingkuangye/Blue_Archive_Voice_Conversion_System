📖 使用教程

Blue Archive Voice Conversion System — 从入门到精通

1概述

这是一个基于 RVC + GPT-SoVits 的变声 Web 应用，可以把你的声音转换成 Blue Archive 角色的声音。

系统分为 4 个主要功能：

功能说明
RVC 变声上传音频 → 选择角色 → 转换声音
人声分离分离音频中的人声和背景音
混响消除去除音频中的混响效果
GSV TTS文字转语音（GPT-SoVits 远程合成）

2首页 & 角色选择

打开网站后首先看到的是首页，所有可用角色按分类展示。

浏览角色 — 页面按分类分组（如「Blue Archive」），每个分组横向滚动显示角色卡片。卡片上显示角色头像、名称、作者和版本号。

搜索角色 — 页面顶部有搜索框，输入角色名、标题或作者即可实时过滤。

进入变声工作室 — 点击任意角色卡片，自动跳转到变声工作室并选中该角色。

3变声工作室 核心功能

这是整个应用的核心页面。

音频输入

来源操作
上传文件点击上传区域选择音频文件（MP3 / WAV / M4A / FLAC）
GSV 音频从 GSV TTS 页面生成的音频会出现在这里（自动切换）
提示：从 GSV TTS 发送过来的音频会显示「GSV 合成音频已就绪」。

转换参数

参数说明建议
升/降调调整音高（-12 ~ +12）+12 男声变女声
检索特征占比保留原音色特征的程度 (0~1)0.7
中值滤波半径削弱哑音 (0~7)≥3 启用
音量包络融合保留原音频音量变化 (0~1)1.0
清辅音保护减少电音 (0~0.5)0.5 不开启
重采样率输出音频采样率0 为不重采样
音频处理算法音高提取算法RMVPE（最佳）

开始转换

选择角色 + 输入音频后，「开始转换」按钮会亮起。点击后等待处理完成，下方会显示播放器和下载按钮。

4音频处理工具 UVR5

人声分离

上传音频 → 选择分离模型 → 点击「开始分离」。结果返回人声和背景音两个音频。

推荐使用 HTDemucs 模型，速度快效果好。
混响消除

上传带混响的音频 → 调整重叠倍数 → 点击「消除混响」。结果返回干声和混响成分。

处理过程中会显示进度条，完成后可直接试听或下载。

5GSV TTS 文字转语音

选择模型

页面按板块分组展示 GSV 模型，点击任意模型卡片进入合成面板。

输入文字

在文本框中输入要合成的文本。单次最多输入 500 个字符。

生成语音

点击「GSV 生成语音」，等待合成完成。结果可在线试听、下载或发送到变声工作室进行 RVC 变声。

6小技巧

  • 批量处理 — 先用 UVR5 分离人声，再把人声发送到变声工作室转换角色音色
  • GSV + RVC 串联 — GSV TTS 合成语音 → 发送到变声工作室 → 选择角色转换，实现文字变角色语音
  • 手机端 — 已适配手机，导航栏可横向滑动，角色卡片左右滑动浏览
  • 搜索功能 — 首页和 GSV 页面都有搜索框，快速定位角色

7常见问题

转换失败/ML 不可用

服务器可能缺少依赖或模型文件，请联系管理员排查。

音频处理一直转圈

长音频需要较长时间。如果配置了远程 UVR5 会更快。超过 2 分钟请重试。

GSV 连接失败

显示「服务不可达」说明远程合成服务器离线，稍后再试。
