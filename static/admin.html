说明

配置远程 RVC 服务后，变声转换和人声分离任务将优先发送到远程服务器处理。

如果远程服务不可用，自动回退到本地处理。

远程服务器需要运行相同的 Blue Archive RVC 服务，并开放 API 端口。

远程服务需设置环境变量 RVC_API_KEY=xxx 来启用 API 鉴权。
